### 1.2 Helper ส่วนตัว `_radius_from_element` (เพิ่มหลัง `_read_pi_table`)

```python
def _radius_from_element(el: alignment.Element) -> float:
    """Return signed design radius for output CSV (0 = tangent)."""
    if el.k_in != 0:
        return 1.0 / el.k_in
    if el.k_out != 0:
        return 1.0 / el.k_out
    return 0.0
```

เหตุผล:
- T: k_in=k_out=0 → return 0
- C: k_in=k_out=1/R → return R (signed)
- SPIN: k_in=0, k_out=1/R → return R (signed)  
- SPOUT: k_in=1/R, k_out=0 → return R (signed)

ค่านี้ตรงกับรูปแบบที่ `_read_alignment` / `parse_alignment_table` ใช้อ่านกลับ → วงจรเขียน-แล้วอ่านกลับได้ (roundtrip-safe)

### 1.3 ฟังก์ชัน `_run_build`

```python
def _run_build(args: argparse.Namespace) -> int:
    """build: PI table CSV → elements_output.csv + controls_so_output.csv."""
    import os
    vertices = _read_pi_table(args.alignment)
    build_result = build_alignment_from_pi(vertices)
    for issue in build_result.issues:
        print(f'warning: {issue}', file=sys.stderr)

    out_dir = args.out_dir if args.out_dir else os.path.dirname(os.path.abspath(args.alignment))
    os.makedirs(out_dir, exist_ok=True)

    # --- เขียน elements_output.csv ---
    el_path = os.path.join(out_dir, 'elements_output.csv')
    el_header = ['StaStart', 'StaEnd', 'N', 'E', 'Azimuth', 'Radius', 'Type', 'Transition']
    el_rows = []
    for el in build_result.elements:
        el_rows.append([
            f'{el.sta_start:.3f}',
            f'{el.sta_end:.3f}',
            f'{el.n:.3f}',
            f'{el.e:.3f}',
            f'{fpmath.rad_to_deg(el.azimuth):.6f}',
            f'{_radius_from_element(el):.3f}',
            el.type,
            el.transition,
        ])
    with open(el_path, 'w', newline='', encoding='utf-8') as f:
        csv.writer(f).writerows([el_header] + el_rows)

    # --- เขียน controls_so_output.csv ---
    cp_path = os.path.join(out_dir, 'controls_so_output.csv')
    cp_header = ['Name', 'STA', 'N', 'E']
    cp_rows = [[cp.name, f'{cp.sta:.3f}', f'{cp.n:.3f}', f'{cp.e:.3f}']
               for cp in build_result.control]
    with open(cp_path, 'w', newline='', encoding='utf-8') as f:
        csv.writer(f).writerows([cp_header] + cp_rows)

    # --- แสดงผลใน terminal ---
    print(f'\n=== Elements ({len(build_result.elements)} rows) -> {el_path} ===')
    print(f'{"StaStart":>10} {"StaEnd":>10} {"N":>12} {"E":>12} {"Az(deg)":>12} {"Radius":>10} {"Type":<6} {"Trans"}')
    print('-' * 90)
    for row in el_rows:
        print(f'{row[0]:>10} {row[1]:>10} {row[2]:>12} {row[3]:>12} {row[4]:>12} {row[5]:>10} {row[6]:<6} {row[7]}')

    print(f'\n=== Control Points ({len(build_result.control)} rows) -> {cp_path} ===')
    print(f'{"Name":<6} {"STA":>12} {"N":>14} {"E":>14}')
    print('-' * 50)
    for row in cp_rows:
        print(f'{row[0]:<6} {row[1]:>12} {row[2]:>14} {row[3]:>14}')

    return 0
```

## 2. Logic flow สรุป

```
smt build <pi_csv> [--out-dir DIR]
    │
    ├─ _read_pi_table(path)          ← มีอยู่แล้วใน cli.py
    │       └─ parse_pi_table(rows)  ← มีอยู่แล้วใน alignment_builder.py
    │
    ├─ build_alignment_from_pi(vertices)  ← มีอยู่แล้ว
    │       └─ BuildResult(elements, control, issues)
    │
    ├─ เขียน  elements_output.csv    (StaStart,StaEnd,N,E,Azimuth,Radius,Type,Transition)
    ├─ เขียน  controls_so_output.csv (Name,STA,N,E)
    └─ พิมพ์ทั้งสองตารางใน terminal
```

# ---- shared PI table fixture (reuse ของ _PI_TABLE ที่มีอยู่แล้ว) ----

def test_build_creates_output_files(pi_csv, tmp_path, capsys):
    """build สร้าง elements_output.csv และ controls_so_output.csv ใน --out-dir"""
    rc = cli.main(['build', pi_csv, '--out-dir', str(tmp_path)])
    assert rc == 0
    assert (tmp_path / 'elements_output.csv').exists()
    assert (tmp_path / 'controls_so_output.csv').exists()


def test_build_element_csv_header(pi_csv, tmp_path, capsys):
    """elements_output.csv ต้องมี header ถูกต้อง"""
    cli.main(['build', pi_csv, '--out-dir', str(tmp_path)])
    with open(tmp_path / 'elements_output.csv', encoding='utf-8') as f:
        header = f.readline().strip()
    assert header == 'StaStart,StaEnd,N,E,Azimuth,Radius,Type,Transition'


def test_build_control_csv_header(pi_csv, tmp_path, capsys):
    """controls_so_output.csv ต้องมี header ถูกต้อง"""
    cli.main(['build', pi_csv, '--out-dir', str(tmp_path)])
    with open(tmp_path / 'controls_so_output.csv', encoding='utf-8') as f:
        header = f.readline().strip()
    assert header == 'Name,STA,N,E'


def test_build_terminal_output(pi_csv, tmp_path, capsys):
    """terminal ต้องแสดงทั้ง Elements และ Control Points"""
    cli.main(['build', pi_csv, '--out-dir', str(tmp_path)])
    out = capsys.readouterr().out
    assert 'Elements' in out
    assert 'Control Points' in out


def test_build_missing_input_returns_exit_1(tmp_path, capsys):
    """ไฟล์ input ไม่มี → exit code 1"""
    rc = cli.main(['build', str(tmp_path / 'no_such.csv'), '--out-dir', str(tmp_path)])
    err = capsys.readouterr().err

## 5. ความเสี่ยงและวิธีป้องกัน

| ความเสี่ยง | ผลกระทบ | วิธีป้องกัน |
|---|---|---|
| **`fpmath` ยังไม่ import ใน cli.py** | `NameError` ตอน run | เพิ่ม `from . import fpmath` ใน import section |
| **Azimuth: radian vs degree** | ค่า Azimuth ใน CSV ผิด | ใช้ `fpmath.rad_to_deg(el.azimuth)` ทุกครั้ง (el.azimuth เก็บ radian) |
| **Radius sign convention** | ผู้ใช้ตีความ L/R ผิด | `_radius_from_element` คืน signed R (+ = right, − = left) ตาม CLAUDE.md |
| **`--out-dir` ไม่มีอยู่** | `FileNotFoundError` ตอนเขียน | `os.makedirs(out_dir, exist_ok=True)` ก่อนเขียน |
| **overwrite output ที่มีอยู่แล้ว** | ข้อมูลเดิมหาย | ยอมรับ (silent overwrite) — เหมาะกับ CLI workflow; ไม่ต้อง guard |
| **ไฟล์ input encoding ไม่ใช่ utf-8** | `UnicodeDecodeError` | `_read_pi_table` ใช้ `encoding='utf-8'` อยู่แล้ว; ปล่อย exception ขึ้น main() จับ → exit 1 |
| **build_result.issues ไม่ว่าง** | ผู้ใช้ไม่รู้ว่ามีปัญหา | print ทุก issue ไปที่ stderr (เหมือน cross-check) |
| **test_build_default_out_dir ปน tmp_path** | tmp_path ≠ pi_csv dir → test ผ่านผิด | ใช้ `pi_csv` fixture ที่ชี้ไปที่ `tmp_path/pi.csv` → out_dir = tmp_path เอง ✓ |
